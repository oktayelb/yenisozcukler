# Yeni Sözcükler 

## Proje Özeti

**Yeni Sözcükler**, Türkçe sözcük türetme etkinliğini arttırmayı amaçlayan Django tabanlı bir web uygulamasıdır.

Platform, içerik kalitesini korumak için gönderim ve moderasyon aşamalarından oluşan bir iş akışı kullanır.

**Canlı Adres:** `yenisozcukler.com`

## Tavsiyeleriniz 

Sitenin UI ve UX kısmının büyük gelişime ihtiyacı var. Bu konuda bir fikriniz varsa lütfen paylaşınız.
